Kashiwazaki SEO Author Panel Manager マニュアル

Person・Corporation・Organization の3種類のエンティティを独立データベースで管理し、著者パネルとSchema.org JSON-LD構造化データを出力するWordPress SEOプラグインです。

E-E-A-T（経験・専門性・権威性・信頼性）はGoogleの検索品質評価において重要な概念です。本プラグインは、著者・企業・組織の情報を構造化データとして正確に出力することで、検索エンジンやAIがコンテンツの信頼性を評価する際の手がかりを提供します。

エンティティ管理

Person・Corporation・Organization の3種類をタブ切替で一元管理

JSON-LD出力

Schema.org準拠の構造化データをStandard/Customモードで自動生成

SEO / GEO における強み

構造化データによるE-E-A-T支援

本プラグインはPerson・Organization・Corporationの構造化データをJSON-LDで出力します。Schema.orgに準拠した著者情報をページに埋め込むことで、検索エンジンがコンテンツの著者・発行者・レビュアーを正確に識別できるようになり、E-E-A-T評価の手がかりとなる可能性があります。

Customモードによる既存スキーマとの連携

Customモードでは、他のプラグインが出力したArticle/NewsArticle/WebPageスキーマの@idを参照し、author/publisher/editorなどのプロパティとしてエンティティを紐づけることができます。これにより、ページ内の構造化データの整合性が向上し、リッチリザルトの表示可能性が高まります。

Core Web Vitals への影響

フロントエンドCSSは約4KBのみで、JavaScriptは一切使用しません（管理画面・ブロックエディタ用のJSを除く）。そのためCore Web Vitals指標に悪影響を与えることはありません。

生成AI（GEO）への対応

JSON-LDで出力された著者・組織の構造化データは、AIがコンテンツの信頼性や著者の専門性を判断する際の機械可読な情報源となります。sameAsプロパティによる外部プロフィールへのリンクは、エンティティの同一性確認に役立ちます。
